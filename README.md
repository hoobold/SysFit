# SysFit

Sistema de gerenciamento para academias, alunos e personal trainers desenvolvido em PHP, MySQL, HTML, CSS e JavaScript.

## Tecnologias

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

## Funcionalidades atuais

* Cadastro de usuários
* Armazenamento em banco de dados MySQL
* Interface responsiva
* Estrutura Mobile First

## Funcionalidades em desenvolvimento

* Login de usuários
* Controle de treinos
* Cálculo de IMC
* Controle de mensalidades
* Dashboard administrativo

## Objetivo

Demonstrar conhecimentos em desenvolvimento web full stack utilizando PHP e MySQL, aplicando conceitos de banco de dados, integração back-end, front-end responsivo e arquitetura de sistemas.
