SYSFIT

O SysFit é um sistema de gerenciamento para academias, alunos e personal trainers, desenvolvido com PHP, MySQL, HTML, CSS e JavaScript.

O objetivo do projeto é centralizar em uma única plataforma funcionalidades essenciais para o ambiente fitness, incluindo cadastro de usuários, controle de treinos, acompanhamento de evolução física, cálculo de IMC e gerenciamento de mensalidades.

Atualmente o sistema conta com funcionalidades de cadastro e armazenamento de usuários em banco de dados MySQL, além de uma interface responsiva desenvolvida com foco em dispositivos móveis (Mobile First).

Funcionalidades planejadas
Cadastro e autenticação de usuários
Painel para alunos, academias e personal trainers
Controle e gerenciamento de treinos
Cálculo automático de IMC
Histórico de evolução física
Controle de mensalidades e pagamentos
Dashboard com métricas e relatórios
Interface responsiva para desktop e dispositivos móveis

Tecnologias utilizadas
PHP
MySQL
HTML5
CSS3
JavaScript
XAMPP
Objetivo

O SysFit nasceu com a proposta de oferecer uma alternativa moderna e intuitiva para a gestão de academias, facilitando a comunicação entre alunos, profissionais e estabelecimentos, além de simplificar o acompanhamento da evolução física dos usuários.
