# SysFit

## Sistema de Gestão Fitness

SysFit é um sistema completo de gestão para academias, desenvolvido em PHP e MySQL.

### Funcionalidades

- **Gestão de Alunos**: Cadastro e controle de alunos
- **Controle de Treinos**: Criação e acompanhamento de programas de treinamento
- **Evolução Física**: Monitoramento do progresso dos alunos
- **Gestão de Personal Trainers**: Controle de profissionais e agendamentos
- **Controle de Mensalidades**: Gerenciamento de pagamentos e inadimplência
- **Dashboard**: Visualização de métricas e relatórios

### Tecnologias

- **Backend**: PHP
- **Banco de Dados**: MySQL
- **Frontend**: HTML, CSS, JavaScript
